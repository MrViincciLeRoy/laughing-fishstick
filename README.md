# Laughing Fishstick
## Overview
Laughing Fishstick is a simple React application that showcases a counter component with increase, decrease, and reset functionality.
## Key Features
* Increase count by 1
* Decrease count by 1
* Reset count to 0
## Tech Stack
* React
* JavaScript
* CSS
* Jest
* @testing-library/react
## Installation
To install the project, run the following command:
```bash
npm install
```
## Usage
To start the application, run the following command:
```bash
npm start
```
## Environment Variables
No environment variables are required to run this application.
## Testing
To run the tests, use the following command:
```bash
npm test
```
This project uses Jest and @testing-library/react for unit testing and integration testing.
